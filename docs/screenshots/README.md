# Screenshots

The root `README.md` embeds three images from this folder. Drop the
following PNG (or JPG — update the paths in the README if so) files here
and they will render automatically on GitHub:

| Filename | What to capture |
|---|---|
| `landing.png` | The public landing page (`/`) with the animated risk-monitor demo |
| `monitoring.png` | The invigilator monitoring dashboard (active sessions + risk column) |
| `risk-modal.png` | The per-session risk modal (score, band, contributing factors, sparkline) |

Recommended width ~1280px, compressed. Until these are added, the
README image slots show their alt text.
